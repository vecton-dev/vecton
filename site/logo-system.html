Clear Space

クリアスペース

ロゴの周囲には必ず最小クリアスペースを確保してください。最小クリアスペースはアークマークの高さ（x）の 1/2 とします。他の要素やページ端に近づけないよう注意してください。

Icon Mark clear = x / 2
Wordmark clear = x / 2

Usage

使用ガイドライン

NG — 変形させない 縦横比を変えてロゴを引き伸ばすことは禁止です。
NG — 色を変えない 承認済みバリアント以外の色で表示しないでください。
NG — コントラスト不足 背景と十分なコントラスト比のない状態での使用は禁止です。
